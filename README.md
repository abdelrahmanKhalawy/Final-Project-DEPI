# SmartClinic System

Multi-Tenant Clinic Management System built using ASP.NET Core and SQL Server.

## Features

- Multi-Clinic (Multi-Tenant) support
- Role-based access control
- Patient management
- Appointment scheduling
- Dashboard analytics

## System Architecture

Layered Architecture:

- Presentation Layer
- Business Logic Layer
- Data Access Layer

## Technologies

- ASP.NET Core
- Entity Framework Core
- SQL Server
- Bootstrap
